# 座席配置システム

> 教室の席替えを楽しいイベントに変える、ブラウザで動く席替えアプリ

スロットマシン形式の抽選、運命の再会・地獄のシャッフル・ラストワンチャレンジなどの特殊演出、Web Audio APIによる効果音、印刷機能まで、席替え当日の盛り上がりに必要なものを一通り揃えています。


---

## 特徴

- **抽選演出**: 名前・列・行の3スロットが順番に止まるドラマ仕立て
- **特殊イベント**: 確率で発動する5種類のイベント
  - 前後列確定演出（教卓前/最後列）
  - 窓際席確定演出（A列）
  - 交換イベント（席が決まった生徒と既存の誰かが入れ替わる）
  - 運命の再会（過去に隣だった生徒同士が再び隣に）
  - 地獄のシャッフル（既に決まった全員の席がランダムに入れ替わる）
- **ラストワンチャレンジ**: 最後の1人になったら座席ルーレットで誰かの席を奪い取る
- **席替え占い**: 席が決まったら100種以上のメッセージから運勢が表示される（カスタム可能）
- **効果音**: Web Audio APIで合成。外部ファイル不要、ON/OFF・音量調整可能
- **男女別座席指定**: 特定の席を男子席/女子席に指定可能、自動チェック付き
- **印刷機能**: B5横向き対応、座席表を生徒数に応じて自動レイアウト
- **データ保存**: localStorageで自動保存、JSONエクスポート/インポートで他PCに移送可能
- **ドラッグ&ドロップ**: 確定後の手動入れ替えにも対応（性別制限チェック付き）


## 使い方

### 初回セットアップ

1. 「名簿設定」をクリック
2. テキストエリアに生徒情報を入力
   - 形式: `番号[Tab]氏名[Tab]性別[Tab]フリガナ`
   - 番号・性別・フリガナは省略可能
   - Excelの該当列をコピペすればOK
3. 「保存」をクリック

### 席替え実行

1. 必要に応じて「座席設定」で使わない席を無効化
2. 「男女設定」で男子席/女子席を指定（必要時のみ）
3. 「カラー設定」で見た目をカスタマイズ
4. 「特殊演出設定」で各イベントの発生確率を調整
5. 「サウンド設定」で効果音をON/OFF
6. **「スタート」をクリック！**
7. 全員の席が決まったら印刷

### よく使う機能

- **教員視点**: 黒板側から見た配置に切り替え（教卓が下に来る）
- **保存/読み込み**: 設定をJSONでエクスポート、別のPCで読み込み可能
- **印刷**: B5横向き、未使用席は自動で省略可能
- **データ全削除**: 全localStorage初期化（注意）

## 技術構成

```
index.html       メインページ（全モーダル含む）
styles.css       全体スタイル（CSS変数で色・サイズを集約）
sound.js         Web Audio APIによる効果音合成
fortune.js       席替え占いメッセージのデフォルトセット
slot.js          スロットマシン本体・特殊演出
print.js         印刷プレビュー・印刷機能
script.js        メインロジック・座席管理・データ永続化
```

依存ライブラリなし。すべてVanilla JavaScript。

### 設計上の工夫

- グリッドサイズは `script.js` の `ROWS=7, COLS=8` 定数で一元管理。教室サイズが変わっても1行修正で対応
- 旧データ形式を自動マイグレーション（7列→8列、ブール値→オブジェクト形式）
- レスポンシブ対応：教室PC・タブレットでも崩れない
- 印刷専用CSSとプレビュー用CSSを分離

## カスタマイズ

### グリッドサイズの変更

`script.js` の冒頭：

```javascript
const ROWS = 7;
const COLS = 8;
```

ここを変えるだけで全体が追従。`styles.css` の `.seating-grid` の `grid-template-columns: repeat(8, ...)` だけは別途変更が必要（CSS側からは定数を参照できないため）。

### 占いメッセージのカスタマイズ

「メッセージ設定」モーダルから「カスタム占いセット」を選択すると、JSONで独自のメッセージセットを読み込めます。形式は `fortune.js` 内の `defaultFortuneSet` を参考に。

### 効果音の変更

`sound.js` の各メソッド（`winChime`, `frontBackChord`, etc.）の周波数や持続時間を変えるだけで音色が変わります。Web Audio APIなので外部音源不要。

### 演出の追加

特殊演出は `slot.js` の `SpecialEffectManager` クラスや `ShuffleEvent` クラスを参考に拡張可能。確率設定は localStorage で管理されているので、新しいスライダーをHTMLに追加して連動させればOK。
